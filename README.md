# GovTrackr : HAU Comelec Candidate Tracking System
This is a project intended for the course Dynamic Webprogramming (6DWEB)


### For the setup of the website: <br>
>1. Clone the repository and extract the archive.
>2. Inside the archive, copy the `govtrackr` folder inside `C:\xampp\htdocs`
>3. Run XAMPP and enable Apache and MySQL.
>4. In the browser type `localhost/phpmyadmin` in the search URL bar.
>5. Go to Import tab and import the `govtrackr_db.sql`.
>6. You may now access the website in the browser by typing `localhost/govtrackr/login.php` in the URL.
